# QuantScreener Pro - A股量化筛选系统

> 30个专业量化策略引擎 | 实时A股数据 | 专业终端风格

![License](https://img.shields.io/badge/license-MIT-blue)
![Platform](https://img.shields.io/badge/platform-Web-green)
![API](https://img.shields.io/badge/API-东方财富-orange)

## ✨ 功能特性

- **30个量化筛选策略** - 涵盖稳定型、激进型、短线、中线、长线、价值投资等多种风格
- **实时A股数据** - 接入东方财富公开API，全市场5000+只股票实时行情
- **专业终端风格** - 暗色主题，仿专业量化交易终端界面
- **大盘指数监控** - 实时显示上证、深证、创业板、科创50、北证50
- **策略详情面板** - 每个策略都有详细的筛选逻辑、适用周期、风险等级、胜率参考
- **数据导出** - 支持CSV格式导出筛选结果
- **API状态监控** - 实时显示数据源、延迟、更新时间

## 🚀 使用方式

### 方式一：直接打开
```bash
# 直接用浏览器打开 index.html 即可
open index.html
```

### 方式二：本地服务器
```bash
# 使用 Python
python3 -m http.server 8080

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8080
```

### 方式三：GitHub Pages
1. Fork 本仓库
2. 进入 Settings → Pages
3. 选择 main 分支，保存
4. 访问生成的链接
